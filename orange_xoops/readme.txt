/* English */

Installation : To install this theme the way it was intended to be installed please upload all contents of the "root" folder (/xoops_thm_Orange_xoops) to your XOOPS_Root.
Finally go to your system preferences and set this theme as your default theme.

IMPORTANT : Orange_xoops theme is morphogenesis theme made for
- Xoops 2.0.18
- Xoops 2.0.18.1
- Xoops 2.0.18.2

For a previous version of Xoops, please download right package at : http://tutos.xoofoo.org/modules/mydownloads/viewcat.php?cid=20




/* french */

Installation : une fois l'archive décompressée, il faut téléverser  le contenu (et uniquement celui-ci) du dossier /xoops_thm_Orange_xoops à la racine de votre site.
Ensuite, dans les préférences de l'administration de votre site, vous sélectionnez ce thème par défaut.
En cas d'utilisation de la surcharge du thème, vous devez cocher dans les préférences de l'administration : l'actualisation du thème et templates à partir du répertoire. 
Une fois le thème utilisé et les fichiers temporaires créés dans le dossier "templates_c" (génération lors de la première visite sur le template "surchargé", vous pourrez décocher cette case.

IMPORTANT : Ce thème est basé sur un gabarit Morphogenesis et ne fonctionne qu'avec 
- Xoops 2.0.18
- Xoops 2.0.18.1
- Xoops 2.0.18.2

Pour l'installer sur une version de Xoops antérieure, veuillez télécharger le pack approprié sur : http://tutos.xoofoo.org/modules/mydownloads/viewcat.php?cid=20
